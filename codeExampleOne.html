Hexagon Shape Example

Hex body

The image on the left displays the code for the hexagon body I used to create on my about page. I have used the class selector to create the main body. I have then nested the image element inside to target the icons of the Languages I have used. I have sized these and used calc to center them inside of the hexagon.

Hex Top

The image on the right displays my use of pseudo-element (::before). This is the same as the bottom of the hex as they were made symmetrically. I used position: absolute so I could position the edges to line up with the body. I then used the borders to create the end of the hexagon utilising the transparent property value. Throughout this project I was using SCSS to create partials and organize my work. As you can see from the $Primary-color. This was used throughout the project, so if I wanted a color swap I would only have to change one color code in one place instead of multiple.
